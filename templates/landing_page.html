Welcome to this demo application built by me. My motivation for creating this application is to put the skills I have learnt into practice and to demonstrate my enthusiasm for coding, showing I have the capabilities required to work as a developer.

ABOUT

The application is based on the Premier League. I chose this domain as football results are fairly familiar to most people and provide enough scope to include elements found in most designs, namely:

  • A clean interface
  • Creation and updating of data
  • Data visualization
  • Feedback - clear messages informing what has happened in response to user action

ARCHITECTURE

  • Web application framework : Flask
  • Client-side : HTML, CSS, Javascript
  • Server-side : Python, Javascript
  • ORM : PeeWee
  • Database : SQLite
  • Architecture Diagram

DOMAIN

Key facts for this domain are:

  • There can be a maximum of 20 teams in the league
  • Each team must have a unique name
  • A team is awarded 3 points for a win , 1 point for a draw and 0 points for a defeat
  • A result is between 2 teams
  • Results cannot be entered unless a minimum of 2 teams exist in the system
  • Half time scoreline cannot be greater than full time scoreline

ENHANCEMENTS

Like every application there's always new ideas that can be implemented to enrich the system. Two such ideas I have are:

  • The ability to set up multiple leagues and seasons
  • Results prediction driven by AI algorithms
