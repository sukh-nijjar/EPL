Welcome to this demo application. My motivation for creating this application is to demonstrate knowledge I have learnt and to show I have the skills and learning mindset to work as a developer. This application contains features commonly required in business applications such as:

  • The creation and management of data using a UI
  • Presentation of information using graphical charts
  • Trapping and presenting errors to ensure system integrity
  • Uploading data in bulk
  • Presenting clear feedback informing what is happening and next steps to take

ABOUT

The application is based on the Premier League. I chose this domain as football results are fairly familiar to most people and provides a set of data that is easy to understand.

Key domain characteristics supported by the system are:

  • There can be maximum of 20 Teams in the league
  • Each team must have a unique name
  • A team is awarded 3 points for a win , 1 point for a draw and 0 points for a defeat
  • A Result is the full-time score between 2 teams
  • Teams must exist in order to create results

ARCHITECTURE

  • Architecture Diagram
  • Web application framework : Flask
  • Client-side : HTML, CSS, Javascript
  • Server-side : Python, Javascript, JQuery
  • ORM : PeeWee
  • Database : SQLite
